A Style-Based Generator Architecture for Generative Adversarial Networks

本篇paper借助style transfer的方法，提出一種新的生成式網路架構，這種新的網路自動學習資料中的high-level attribute，同時也學到生成資料的隨機性。結果顯示這樣的模型可以達到更好的差值特性，並更能表述淺在特徵的變異分佈。

Learning Linear Transformations for Fast Arbitrary Style Transfer

先前做任意風格轉換的演算法，通常計算複雜度都很高，而且不容易捕捉到畫風之間的變異性，也會產生一些artifact。本篇paper提出一種transformation matrix的形式來解決arbitrary style transfer的問題，這種彈性的架構在轉換的過程中也會將content給保留住。

Attention Is All You Need

目前的翻譯模型大多基於很複雜的遞迴神經網路或卷積神經網路，並搭配上encoder-decoder架構。表現好的模型通常添加了注意力機制。本paper提出了Transformer，是一個僅依賴注意力機制的模型，完全不使用卷積或是遞迴機制。實驗證明，這樣的網路架構可以平行化的訓練並減少訓練時間，在英德翻譯取得了28.4 BLEU的成績

Image Inpainting for Irregular Holes Using Partial Convolutions

在inpainting的方法中，傳統的方法都是使用標準的卷積層來做，大概想要達到的目的是：給定一張含有空洞的圖片，求取完整圖片的似然估計。但這樣的作法通常會產生一些artifact或模糊的情形。當然也可以使用後處理來消除這些現象，不過效果並不好，甚至可能失敗。
